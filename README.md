# Om Bhatt - Personal Portfolio

This repository contains the source code for my personal academic and professional portfolio. It serves as a central hub for my work as a Robotics Researcher and Engineer, showcasing my projects in autonomous navigation, CAD engineering, and embedded systems.

The site features a custom-built, simulated hardware bootloader, dynamic UI micro-interactions, and a responsive design tailored for tech and research.

🌐 **Live Site:** [https://ombhatt2.github.io/]

---

## 🛠️ Tech Stack & Features

* **Framework:** [Jekyll](https://jekyllrb.com/) hosted on GitHub Pages.
* **Core Theme:** Forked and heavily customized from [Academic Pages](https://academicpages.github.io/).
* **Styling & UI:** Custom CSS/HTML, featuring dark-mode elements and glass-morphism effects.
* **Animations:** * `AOS.js` for scroll-triggered fade-ups.
  * `Vanilla-tilt.js` for 3D interactive project cards.
  * `Particles.js` for the interactive background network.
* **Custom Interactions:** Bespoke hardware-diagnostic loading screen and custom cursor tracking.
